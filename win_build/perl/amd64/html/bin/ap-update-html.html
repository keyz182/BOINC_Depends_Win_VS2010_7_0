NAME

ap-update-html - Regenerate any out-of-date HTML

SYNOPSIS

ap-update-html [ --force ] [ --verbose ]

DESCRIPTION

If new modules has been installed then they might not have had their documentation converted to HTML yet. This script will bring the HTML up-to-date with what modules are installed.

The following command line options are recognized:

--force

Force HTML documents to be regenerated even if they appear to be up-to-date.

--verbose

Print noise about what's done while running.
